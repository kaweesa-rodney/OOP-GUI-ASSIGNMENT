# Django Web App + Tkinter Desktop App

This repository contains **two separate Python applications**:

1.  **Django Web Application** — a web-based app that runs in your browser  
2.  **Tkinter Desktop Application** — a native desktop GUI app  

---

## Quick Start Guide

### Django Web Application

The Django web app is powered by a self-contained setup script called `run.py`.  
It automatically handles:
- Creating a virtual environment (`venv/`)
- Installing dependencies from `requirements.txt`
- Starting the development server

#### **Windows**
python run.py

#### **MacOS**
python3 run.py


## Quick Start Guide

### Tkinter desktop Application

The Desktop app is powered by a self-contained setup script called `run_gui.py`.  Redirect to the GUIControls folder
It automatically handles:
- Creating a virtual environment (`venv/`)
- Installing dependencies from `requirements.txt`
- Starts the application(opens the gui)

#### **Windows**
python run_gui.py

#### **MacOS**
python3 run_gui.py
